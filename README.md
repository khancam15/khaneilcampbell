# Khaneil Campbell — Cybersecurity Portfolio

This repository contains a simple static portfolio site for Khaneil Campbell, focused on cybersecurity labs, SOC automation, incident response, and professional branding.

## Files

- `index.html` — main landing page for the portfolio
- `.gitignore` — ignores common macOS, editor, and build files

## Notes

- The page is styled with inline CSS for a polished dark theme
- Includes sections for featured labs, proof, about, and contact
- Links include GitHub, LinkedIn, and email contact

## Deploy

This is a static HTML site, so you can deploy it with GitHub Pages or any static hosting provider.

### GitHub Pages

1. Push the repository to GitHub
2. In repository settings, enable GitHub Pages from the `main` branch
3. Visit the published site URL provided by GitHub

## Development

To update the site:

1. Edit `index.html`
2. Save your changes
3. Commit and push to GitHub

```bash
git add index.html README.md .gitignore
git commit -m "Add README and project documentation"
git push origin main
```
